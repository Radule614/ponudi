# Ponudi

A e-commerce website based on olx.ba.
The project was primarily meant to be a uni project for learning Angular. I was working on the frontend part while having my friend on the backend.


Technologies used:
  - Angular
  - SCSS
  - Angular Material UI
  - NestJS
  - Supabase
